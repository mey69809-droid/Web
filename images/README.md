# Images Folder

This folder should contain all the images referenced in the website.

## Quick Image List

For development purposes, you can use placeholder images or download royalty-free images from:
- Unsplash (unsplash.com)
- Pexels (pexels.com)
- Pixabay (pixabay.com)

Search for terms like:
- "Angkor Wat Cambodia"
- "Phnom Penh Cambodia"
- "Siem Reap Cambodia"
- "Cambodia beaches"
- "Cambodian food"
- "Cambodian culture"

## Image Dimensions Recommendations

- **Hero background**: 1920x1080px or larger
- **Card images**: 800x600px or similar aspect ratio
- **Detail images**: 1200x800px or similar
- **All images**: Optimize for web (JPEG format recommended, 80-90% quality)
