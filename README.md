# Travel Places in Cambodia

A modern, responsive, and user-friendly website showcasing the beautiful destinations and rich culture of Cambodia. Built with HTML5, CSS3, and Bootstrap 5.

## Features

- **10 Complete Pages**: Home, Destinations, Angkor Wat, Phnom Penh, Siem Reap, Beaches & Islands, Food & Culture, Travel Tips, About Cambodia, and Contact
- **Responsive Design**: Fully responsive for mobile, tablet, and desktop devices
- **Cambodian-Inspired Design**: Color scheme featuring warm golds, reds, greens, and natural tones
- **Sticky Navigation**: Consistent navigation bar with active page highlighting
- **Modern UI/UX**: Clean layout, smooth hover effects, and professional tourism-style appearance
- **Bootstrap 5**: Utilizes Bootstrap grid system, cards, buttons, badges, and utilities
- **Semantic HTML**: Proper use of semantic HTML5 elements

## Getting Started

1. **Add Images**: Place all required images in the `images/` folder. You can use placeholder images or download royalty-free images of Cambodia from sources like Unsplash, Pexels, or similar.

2. **Open in Browser**: Simply open `index.html` in your web browser to view the website.

3. **Local Development**: For best results, use a local web server:
   - Python: `python -m http.server 8000`
   - Node.js: `npx http-server`
   - VS Code: Use the Live Server extension

## Customization

To customize the website:

1. **Colors**: Edit CSS variables in `css/style.css` (root section)
2. **Content**: Update HTML content in respective page files
3. **Images**: Replace images in the `images/` folder
4. **Navigation**: Update navigation links in all HTML files (they're consistent across pages)

## Notes

- All pages include consistent navigation and footer
- Active page highlighting is handled via the `active` class on navigation links
- The website is fully static (no backend required)
- Form on contact page is for display (no form processing included)
